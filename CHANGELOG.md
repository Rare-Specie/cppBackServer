## 📥 升级指南

### 从 v1.1.0 升级到 v1.2.0

1. **备份数据**
   ```bash
   cp -r data/ data_backup/
   ```

2. **更新代码**
   ```bash
   git pull origin main
   ```

3. **重新编译**
   ```bash
   ./build.sh
   ```

4. **验证运行**
   ```bash
   ./main
   ```

### 从 v1.0.0 升级到 v1.1.0

1. **备份现有数据**
2. **更新依赖**
   ```bash
   brew install crow nlohmann-json
   ```
3. **重新编译**
4. **测试所有功能**
